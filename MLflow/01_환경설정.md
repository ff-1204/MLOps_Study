### 편의를 위한 변수 저장

매번 긴 경로를 입력하기 불편하니 PowerShell 변수로 저장해두자.
아래 4줄을 **터미널을 열 때마다** 입력한다.

```powershell
$python = "C:\Users\myesu\miniconda3\python.exe"
$mlflow = "C:\Users\myesu\miniconda3\Scripts\mlflow.exe"
$env:PYTHONUTF8 = "1"
$env:PATH = "C:\Users\myesu\miniconda3\Scripts;" + $env:PATH
```

> `PYTHONUTF8=1` — MLflow 자동 로깅(`autolog`)이 Windows CP949 인코딩 오류 없이 동작하게 한다.
> `PATH` 설정 — `mlflow models serve` 내부에서 uvicorn을 찾지 못하는 오류를 방지한다.
> **주의:** 터미널을 닫고 새로 열면 초기화된다. 다시 입력해야 한다.

## 4. MLflow UI 첫 실행

MLflow를 설치하면 웹 대시보드를 바로 열 수 있다.

```powershell
& $mlflow ui
```

아래 메시지가 나오면 정상이다.

```
INFO:     Uvicorn running on http://127.0.0.1:5000 (Press CTRL+C to quit)
```

브라우저에서 `http://localhost:5000` 접속

> **포트 충돌 오류가 나면?**
> 5000번 포트를 다른 프로그램이 쓰고 있는 것이다.
> ```powershell
> & $mlflow ui --port 5001
> ```
> 그러면 `http://localhost:5001` 로 접속하면 된다.
